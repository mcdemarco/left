These bookmarklets make it easier to create palettes and patterns on the COLOURlovers website. You don't need to be logged in to Coolors or ColourLovers to use these bookmarklets, but you will need to be logged into ColourLovers to save palettes or patterns. Whether or not you're logged in, palettes and patterns will not save or publish automatically; you will need to do that step, as well as provide titles.

Palette Creation

Both bookmarklets are designed to import Coolors palettes to ColourLovers. To use the basic bookmarklet, "Palettize", select your palette at Coolors and click on the bookmarklet. This will open up a ColourLovers window or tab (depending on your browser settings) with your colors filled in to the basic palette creator. All you need to do is name and save the palette.

To use the advanced bookmarklet, "Copasetic", select your palette at Coolors and copy the full Coolors URL. Then open Copaso. Click on the bookmarklet to open a popup, and paste the Coolors URL into the popup. Nothing obvious will happen, but you can now click on each blank color swatch and hit the return key (for a total of five times) to get your colors. Then you can adjust your palette widths, enter a title, and save.

Pattern Creation (from a pattern and a palette)

To use the pattern creation bookmarklet, "P&P", open the COLOURlovers page for the palette you want to use to color a pattern and click on the bookmarklet. Do the same on the page for the pattern or pattern template you want to color. If you have selected both a palette and a pattern in the current browser, the bookmarklet will open up a link to create that pattern from that palette.

The bookmarklet stores both your pattern and your palette, so the next time you click the bookmarklet on, say, a pattern, it will combine it with the last palette you used. If the bookmarklet fails mysteriously, check the license on the palette or pattern you clicked on; it may not permit reuse.

Install the Bookmarklets

To install the basic bookmarklet, drag this link to your Favorites or Bookmarks bar: Palettize.

To install the Copaso bookmarklet, drag this link to your Favorites or Bookmarks bar: Copasetic.

To install the pattern creation bookmarklet, drag this link to your Favorites or Bookmarks bar: P&P.

Bookmarklets on iOS

To install any bookmarklet directly on an iPhone or iPad, you can follow these instructions on adding a bookmarklet to Mobile Safari, but I find it's easier to add the bookmarklet to desktop Safari and let iCloud sync it. (Syncing can be set up under Settings | iCloud | Safari on iOS, and System Preferences | iCloud | Safari on MacOS.)

Note that the Copasetic and P&P bookmarklets wil not be much use on an iPhone, iPad, or other touch screen because the ColourLovers tools are not very responsive. The Palettize bookmarklet will work, but you may have some trouble finding the popup to name your new colors (if your palette happens to include new colors). Try pinching to find it.
