LaTeX is a set of macros for TeX. LaTeX is available for all operating systems, in a bewildering variety of implementations. For the beginner, I recommend either LyX or the following popular combinations of an underlying TeX engine and a nice front-end and/or text editor:

  • For Windows, MiKTeX and WinEdt.
  • For MacOS X, TeXLive-teTeX and TeXShop
  • For Unix systems, emacs and TeX should come pre-installed, but you might want to look into TeX Live.

Some of these packages may use A4 as their default paper size. See the troubleshooting section of the sffms manual for more details.

The sffms manual also includes an introduction to LaTeX sufficient for use with sffms, but here are some further LaTeX references:
