These are old versions of sffms, provided in case the current one breaks for some people. Are you sure you don't want the latest version?

Version history

Version 2.0 included some publisher-specific options (for DAW, Baen and the Writers of the Future contest), optional use of geometry.sty, and extensive troubleshooting information.

Version 1.3 included a rudimentary word counter and even more documentation.

Vesion 1.2 included extended documentation and added control over the page headers.
